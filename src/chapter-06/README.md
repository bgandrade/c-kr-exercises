# Chapter 06

**Title:** Structures

**Exercises:** 6

## Exercises

- [ ] `6-01` - Our version of `getword` does not properly handle underscores, string constants, comments, or preprocessor control lines. Write a better version.
- [ ] `6-02` - Write a program that reads a C program and prints in alphabetical order each group of variable names that are identical in the first `n` characters but different somewhere thereafter. Do not count words within strings and comments. Make `n` a parameter that can be set from the command line.
- [ ] `6-03` - Write a cross-referencer that prints a list of all words in a document, and for each word, a list of the line numbers on which it occurs. Remove noise words like "the", "and", and so on.
- [ ] `6-04` - Write a program that prints the distinct words in its input sorted into decreasing order of frequency of occurrence. Precede each word by its count.
- [ ] `6-05` - Write a function `undef` that will remove a name and definition from the table maintained by `lookup` and `install`.
- [ ] `6-06` - Implement a simple version of the `#define` processor suitable for use with C programs, based on the routines of this section. You may also find `getch` and `ungetch` helpful.
